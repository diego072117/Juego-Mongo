Para crear una base de datos deberas utilizar el siguiete codigo 1

+

use nombre de Base de Datos.

Si quieres crear una coleccion puedes utilizar el siguiente codigo 2

+

db.createCollection("Nombrecoleccion")

Si deseas insertar un documento en una coleccion con este codigo lo podras hacer: 3

+

db.coleccion.insert({propiedad:"valor",propiedad;"valor"})

Ahora si deseas insertar varios documentos puedes hacerslo asi: 4

+

db.coleccion.insert([{propiedad:"valor",propiedad;"valor"},{propiedad:"valor",propiedad:"valor"}])

Si necesitas consultar las colecciones creadas tienes que colcocar este codigo: 5

+

show collections

Si quieres ver los documentos que creaste en una coleccion podras usar este codigo: 6

+

db.coleccion.find()

Ahora si quieres consultar el documento por un valor en especifico lo puedes hacer asi: 7

+

db.coleccion.find({file:"valor"})

Y si quieres ver los documentos consultados de manera organizada debes poner este codigo: 8

+

db.coleccion.find().pretty()

Si deseas actualizar un file de tu documento lo puedes hacer de esta manera: 9

+

db.coleccion.update({'file':'valor'},{$set:{'file':'valor'}})

Si quieres eliminar un documento lo puedes hacer asi: 10

+

db.coleccion.deleteOne({'file':'valor'})

Y por ultimo si deseas eliminar la base de datos puedes utilizar este codigo: 11

+

db.dropDatabase()
